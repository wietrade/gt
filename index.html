终端命令行

支持终端命令行 git clone , wget , curl 等工具下载.
支持 raw.githubusercontent.com , gist.github.com , gist.githubusercontent.com 文件下载.
注意：不支持 SSH Key 方式 git clone 下载.

git clone

git clone https://x.haod.me/ https://github.com/stilleshan/ServerStatus

git clone 私有仓库

Clone 私有仓库需要用户在 Personal access tokens 申请 Token 配合使用.
git clone https://user:your_token @x.haod.me/ https://github.com/your_name/your_private_repo

wget & curl

wget https://x.haod.me/ https://github.com/stilleshan/ServerStatus/archive/master.zip
wget https://x.haod.me/ https://raw.githubusercontent.com/stilleshan/ServerStatus/master/Dockerfile
curl -O https://x.haod.me/ https://github.com/stilleshan/ServerStatus/archive/master.zip
curl -O https://x.haod.me/ https://raw.githubusercontent.com/stilleshan/ServerStatus/master/Dockerfile
